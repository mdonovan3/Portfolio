1. What I Can Help With in a Lab

Research labs often run on fragile workflows: manual REDCap downloads merged in Excel, ad hoc CSV joins that break when a column name changes, no dashboards, no reproducible pipelines, no one comfortable designing a proper schema. The result is that a PI or coordinator spends real time on data maintenance instead of research.

The skills below map directly to those problems. They were built in a different domain (restaurant operations), but the data challenges are the same: messy sources, inconsistent formats, missing keys, files that don't join cleanly, reports that need to update automatically rather than requiring a manual pull every week.
